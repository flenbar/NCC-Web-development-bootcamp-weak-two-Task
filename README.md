- 🕹️ DOM manipulation
- 📱 Responsive web design principles

## 🛠️ Getting Started

Follow these instructions to get the project running on your machine.

### 🔗 Prerequisites

- [Git](https://git-scm.com/) 🐙
- [Node.js & npm](https://nodejs.org/) 🟢 (if JavaScript tooling is required)
- Code editor (such as [VSCode](https://code.visualstudio.com/)) 📝

### 💾 Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/flenbar/NCC-Web-development-bootcamp-weak-two-Task.git
    cd NCC-Web-development-bootcamp-weak-two-Task
    ```
2. (If applicable) Install dependencies:
    ```bash
    npm install
    ```

### ▶️ Running the Project

- **Static HTML, CSS, JS:**  
  Open `index.html` (or relevant file) directly in your web browser 🌍.

- **With a Dev Server:**  
  If using tooling like [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) in VSCode, right-click the HTML file and select **"Open with Live Server"** 🚦.

- **Node.js Apps:**  
  Start the back-end (if included) by running:
  ```bash
  npm start
  ```
